# simple-phishing-detector
This project is for detection of phishing url's, implemented in python using regrex pattern , and uses url feature extraction

Features:
1)Identify urls that use ip addressing insted of domain names.
2)Detect suspicious phishing keywords
3)Analyze subdomains patterns
4)check unusually long URLs.
5)Generate a risk score based on the above indicators

Technology:
1)Python
2)Regex

Note:
This is only for educational purposes and not be used as a complete phishing detection system.
